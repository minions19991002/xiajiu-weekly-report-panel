# 下酒周报看板：公网部署版

这个版本适合发给多人使用。部署完成后，所有人只需要打开同一个公网链接，不需要解压、不需要启动本地服务、不需要打开 127.0.0.1。

## 推荐方式

部署到支持 Docker 的云平台，例如 Render、Railway、Fly.io 或自己的云服务器。

## Render 部署步骤

1. 打开 Render 并登录。
2. 新建 Web Service。
3. 选择 Docker 部署。
4. 上传或连接本部署包代码仓库。
5. 部署完成后，平台会给一个公网链接。
6. 把公网链接发给别人即可使用。

## 使用方式

打开公网链接后：

1. 一次上传 10 个必需 Excel 文件。
2. 可额外上传上周已生成周报，用于延续 Sheet7 趋势。
3. 根据需要调整“饿了么进店提升换算系数”。
4. 点击生成并下载完整周报。

## 说明

- 本版本监听云平台提供的 PORT。
- 上传文件只用于当次生成，输出文件保存在服务器临时目录。
- 生成的 Excel 会隐藏：业绩趋势、26年利润额和食亨服务费、门店明细。
- 输出文件名按本期周期自动命名，例如：【下酒】0615-0621周报数据.xlsx。

## 本地测试

```bash
pip install -r requirements.txt
python fill_report_dashboard.py
```

然后打开：

```text
http://127.0.0.1:8787/
```
